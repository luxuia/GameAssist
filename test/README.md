# Dota 2 Test Screenshots

此目录用于存放测试用的 Dota 2 游戏截图。

## 如何获取测试图片

### 方法 1: 使用 Steam 截图功能
1. 在 Dota 2 中按 F12 截图
2. 在 Steam 客户端中查看 > 截图 > Dota 2
3. 选择截图并右键 > "在文件夹中显示"
4. 复制截图到本目录

### 方法 2: 使用 Windows 截图工具
1. 运行 Dota 2 游戏
2. 按 Win + Shift + S 打开截图工具
3. 截取游戏画面
4. 保存到本目录

## 建议的测试场景

为了全面测试游戏辅助功能，建议收集以下类型的截图：

1. **对线阶段** - 早期游戏，兵线对线
2. **团战场景** - 多英雄交战的混乱场面
3. **商店界面** - 物品购买界面
4. **小地图** - 包含小地图信息的完整画面
5. **英雄信息** - 技能、属性面板可见
6. **游戏结束** - 胜利/失败结算界面
7. **不同时间段** - 白天/夜晚
8. **不同地图区域** - 上路/中路/下路/野区
9. **不同英雄界** - 天辉/夜魇
10. **战斗状态** - 血量/蓝量变化

## 图片要求

- **格式**: PNG, JPG, JPEG, 或 BMP
- **分辨率**: 1920x1080 或更高（与实际游戏分辨率匹配）
- **内容**: 完整的 Dota 2 游戏窗口截图

## 测试步骤

1. 将测试图片放入此目录
2. 启动 GameAssist 应用
3. 点击 "Upload Screenshot" 按钮
4. 选择测试图片
5. 观察 AI 分析结果

## 注意事项

- 请确保截图不包含敏感信息（如聊天记录中的个人信息）
- 使用自己游戏中的截图或官方公开的截图
- 避免使用可能涉及版权问题的图片
